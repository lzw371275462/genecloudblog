##  2.1 流程部署须知

### 1. 参数类型
&emsp; 流程部署中需要选择各种参数类型，其中包括:样本清单, 样本分组, 对比分组, 组合文件, 文件, 数字, 文本, 单选等, 各类型说明如下：

```
1. 样本清单: 标准格式的fq配置格式，不做赘述
    python 传参机制： python {check.py}  -i {样本清单}

2. 样本分组: 单个分组，比如：groupA	Sam1,Sam2,Sam3
    python 传参机制： python {check.py}  -i {样本清单},{样本分组} 
	
3. 对比分组: 多个对比分组，比如：Case	Sam1,Sam2,Sam3	Control	Sam4,Sam5,Sam6
    python 传参机制： python {check.py}  -i {样本清单},{对比分组} 
	
4. 组合文件:  没有固定格式，可将excel 转成txt
    python 传参机制： python {check.py}  -i {样本清单},{组合文件} 

5. 文件：普通txt文件，不做任何检查，用户传什么就是什么

6. 数字：整数，好像不支持小数，小数请改成文本

7. 文本: 所有字符串都可以

8. 单选: 设置多个可选项，只能从中选择一个
```

&emsp; 注意：样本分组|对比分组|组合文件 都是分别和样本清单捆绑的，所以每个检查脚本的输入不会超过两个。
